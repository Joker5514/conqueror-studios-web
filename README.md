# Conqueror Studios Web

Official Conqueror Studios portfolio site for Randy Jordan and Conqueror Studios.

## Features

- Futuristic dark red / neon landing page
- Project showcase for current products
- Waitlist signup form UI
- Feature suggestion form UI
- Donations/support section
- Contact section
- Responsive layout
- Zero-build static deployment

## Structure

- `index.html` — single-page site
- `styles.css` — visual system and responsive styles
- `script.js` — interactions, form handling, animated counters, nav behavior

## Deploy

This repo is static and can be deployed directly to GitHub Pages, Vercel, Netlify, or any static host.
