## What this is

This is the marketing and research-facing website for **Conqueror Studios**, an independent AI R&D lab focused on:

- **Federated agentic architectures** (AI Bridge, MCP-connected agent networks)
- **Multi-agent orchestration** (OrchestrAI Nexus, LangGraph cyclic state machines)
- **Voice-first interfaces** (VoiceIsolate Pro, voice micro-surveys)
- **Evaluation and observability** for production AI systems

Every project is framed as a research experiment — a question, a minimal testable system, and instrumentation.

cd conqueror-studios-web

# No build step required — pure HTML/CSS/JS
# Just deploy the root directory to Vercel or GitHub Pages

**Vercel settings:**
- Framework: `Other`
- Root directory: `/`
- Output directory: `/`
- Build command: *(none)*
