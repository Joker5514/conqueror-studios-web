The Lab

How Conqueror Studios operates

Start from a research question

Every engagement begins with a framed question about behavior we want to observe or change — not a spec sheet or a list of features.

Build the smallest testable system

We build the thinnest vertical slice that exercises real data and real users, then instrument it before iterating.

Observe, not guess

Traces, eval harnesses, and behavioral telemetry turn agent behavior into something you can debug, diff, and improve — not just prompt-tune blindly.

Prefer cyclic graphs over pipelines

Real AI workflows require loops for reasoning, error detection, and self-correction. We architect with LangGraph cyclic state machines rather than linear DAGs.

Working Principles
  • Start from real constraints, not polished briefs.
  • Prefer observability over clever prompts.
  • Protocol-first: MCP as the universal connector layer.
  • Federated intelligence over monolithic “God Models.”
  • Every deployment is a long-running experiment.
  • Build tools that developers actually want to use.
Research Areas
  • Multi-agent coordination strategies and failure modes.
  • Voice and ambient interface interaction design.
  • Evaluation harnesses for production AI behavior.
  • Federated agentic architecture and MCP tooling.

  • Forensic psychology AI and behavioral analysis.
  • Autonomous workflow and CI/CD intelligence.
Lab Instruments

Stack & infrastructure

We care about how tools behave under stress. Our stack is biased toward transparency, composability, and developer ergonomics — because experiments are messy by default.

Orchestration & agents

  • LangGraph Stateful cyclic graph orchestration with checkpoints and time-travel debugging
  • Model Context Protocol Universal tool connector layer across all agent stacks
  • React Flow Visual graph UI — agents rendered as observable nodes, not black boxes
  • E2B Firecracker VMs Sub-second sandboxed execution environments for code verification

Web & app runtime

  • Next.js 15 App Router, RSC streaming, and MCP proxy API routes
  • TypeScript Full type safety across agent schemas, tool definitions, and UI
  • Tailwind CSS Utility-first design system for fast, consistent UI iteration
  • Vercel Serverless deploys, edge functions, and automatic preview environments

Data & persistence

  • PostgreSQL + pgvector Relational state, LangGraph checkpoints, and embedding storage
  • Supabase Managed Postgres with real-time subscriptions and auth
  • Redis Job queues for long-running agent workflows beyond serverless timeouts

AI providers & models

  • OpenAI GPT-4o Primary reasoning and code generation workloads
  • Claude 3.5 Sonnet Long-context synthesis and deep research nodes
  • Gemini 2.5 Flash Voice synthesis, fast inference, and TTS experiments
  • Brave Search MCP Live web retrieval for research agent loops

Observability & eval

  • LangSmith Trace-level visibility into agent decisions and tool calls
  • Vercel Analytics Real-user performance and Core Web Vitals
  • Custom eval harnesses Behavioral regression tests and offline comparisons against baselines

Audio & voice

  • Web Audio API Browser-native audio pipeline for capture and processing
  • xterm.js Browser-based terminal mirroring WebSocket streams from E2B VMs
  • Gemini TTS Voice synthesis for conversational and survey interfaces
