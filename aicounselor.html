Therapeutic Framework

Seven therapeutic modalities

The system utilizes seven therapeutic modalities and real-time crisis detection for immediate 988/Lifeline intervention. Each session is adaptive — the counselor shifts modality based on detected emotional tone and session history.

  • GPT-4o Core — Primary reasoning and empathetic response generation.
  • Whisper Transcription — Voice-to-text for spoken session input.
  • Crisis Detection — Real-time trigger monitoring with 988/Lifeline integration.
  • Animated Avatar — Framer Motion avatar with emotion-mapped state machine.
  • Credit System — 100-credit threshold with transparent session handoff.
